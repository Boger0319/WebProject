首先

在你的电脑中安装 Python ，此处略过...

在你的电脑中安装 xlwing 库，按住 win+R 键召唤出运行框,并输入 cmd 回车召唤出命令行

在命令行中输入 pip install xlwings 进行安装。

你现在已经成功了30%

基础语法

python 基础

    首先让我们简单学习一下Python的基础语法，方便下面xlwings的使用
  • 列表（数组）
list = ['1','3','4','5'] # 中括号包裹数据
list.append('6') # 在数组末尾添加新数据
list.insert(1,'2') # 在自选位置添加新数据
  • 条件语句
# 单条判断
          if (表达式) :
          语句1
          #多条件判
          # “与”为and
          # “或”为or
          if (表达式1) and (表达式2) or (表达式3):
          语句2
          # 或者
          if (表达式1) :
          if (表达式2):
          语句2
          # 多情况判断
          if (表达式1) :
          语句3
          elif (表达式2) :
          语句4
          else :
          语句5
  • 循环语句
>  for 循环
          for 变量 in 集合 :
          语句
          >  while 循环
